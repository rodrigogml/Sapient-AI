# Agentes Codex

Esta pasta agrupa definições de agentes customizados para Codex.

Use este espaço para agentes independentes das skills ou para composições que coordenem múltiplas skills do repositório.

## Agentes Disponíveis

- [`AGENTS-DATABASE.md`](AGENTS-DATABASE.md): instruções para criação e alteração de modelos, scripts e objetos de banco de dados.
- [`AGENTS-DELEGATOR.md`](AGENTS-DELEGATOR.md): ponto de entrada para decidir quais instruções especializadas devem ser lidas em cada tarefa.
- [`AGENTS-DEVELOPMENT-CYCLE.md`](AGENTS-DEVELOPMENT-CYCLE.md): instruções para início, execução, encerramento, commit e push de ciclos de desenvolvimento.
- [`AGENTS-JAVA-CODING.md`](AGENTS-JAVA-CODING.md): instruções de escrita, nomenclatura, Javadoc, enums, objetos e testes para código Java.
- [`AGENTS-JAVA-SPRING-BOOT.md`](AGENTS-JAVA-SPRING-BOOT.md): instruções para organização e configuração de repositórios Java baseados em Spring Boot.
